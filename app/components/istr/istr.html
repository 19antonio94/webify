HOW TO----->

  • info_outlineHow to install:

    First clone and install dependency

    git clone git@github.com:AD2014/webify
    cd webify
    npm install


    the npm install will run bower install and gulp build to generate all the html for the app in the build directory.

  • addHow to test:

    npm test


  • addHow to develope:

    gulp watch


  • addHow to change env:

    export NODE_E
    npm install

    ## OR
    export NODE_ENV=staging
    npm install

    ## OR
    export NODE_ENV=devel
    npm install


  • addhow to clean and rebuild all:

    npm run-script clean
    npm install
